# Notes:
Command line options for the app:
```
Options:
      --help     Show help                                             [boolean]
      --version  Show version number                                   [boolean]
  -b, --base     Base of the multiplication table            [number] [required]
  -l, --list     Show the multiplication table in the console
                                                      [boolean] [default: false]
  -t, --limit    Limit of the multiplication table        [number] [default: 10]
  ```
